# P1 上线后 10 分钟验收清单（增长层）

> 目标：每次上线后快速确认“归因 + 邀请 + 跟进”三件事没断。

## 1) 归因参数可落库
- 用带参数的链接进入 Bot（例如带 `utm_campaign` 或 `source/campaign`）
- 预期：bridge/事件中能看到参数，并能在 CRM 线索中查询到

## 2) 邀请关系可追踪
- A 用户生成邀请链接，转发给 B
- B 用该链接进入并完成建档
- 预期：后台能看到 `inviter -> invitee` 关系；奖励只发一次

## 3) CRM 跟进可执行
- 对一个“未支付订单/未确认预约”生成跟进任务
- 一键发送 Telegram 模板
- 预期：触达日志写入；跟进结果可记录

## 4) 追单频控有效
- 对同一用户触发两次追单
- 预期：24h 内只发送一次（或第二次被阻断并记录原因）
