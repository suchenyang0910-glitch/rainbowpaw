# MiniApp 与商户端三语改造-页面设计

## 全局设计（跨页面）

### Layout
- 商户端（Desktop-first）：内容区最大宽度 1200px，左右留白自适应；表单采用 CSS Grid（2 列/3 列可折叠）+ 局部 Flex。
- MiniApp：以单列堆叠为主；但在平板/桌面容器下保持 600–800px 居中。

### Meta Information
- 商户端页面 Title：`{页面名} - 多语言`（示例：`商品编辑 - 多语言`）
- Description：说明该页用于语言切换/多语内容维护。
- Open Graph：沿用现有站点默认设置，仅更新 title。

### Global Styles
- 基础背景：#FFFFFF；页面浅灰底（可选）：#F7F8FA
- 主题色：沿用现有品牌色（用于主按钮、选中 Tab、开关）
- 字体：正文 14–16px；标题 18/20/24px 分级
- 按钮：Primary（实心）/Secondary（描边）；hover 提升亮度 4–6%
- 表单错误态：红色提示 + 输入框边框高亮；缺失语言提示用橙色“提醒”级别

## 1) 商户端-系统设置/语言

### Page Structure
- 右侧主内容（卡片容器）+ 顶部面包屑（如现有有）。

### Sections & Components
1. 顶部标题区
   - 标题：系统设置 / 语言
   - 说明文案：语言切换仅影响界面显示与商品内容展示语言偏好。
2. 语言选择卡片
   - 组件：Radio Group 或 Select（3 个语言选项）
   - 行为：切换后立即生效（UI 文案刷新）；右下角 Toast 提示“已切换到xx”。
3. 默认语言与回退策略说明卡片
   - 文案：
     - “默认语言”为商品必填校验依据
     - “展示语言”缺失时自动回退到默认语言
4. 保存区
   - 若采用“即时生效 + 自动保存”：显示最后保存时间
   - 若采用“编辑后保存”：底部固定操作条（保存/取消）

### Responsive
- < 768px：卡片全宽堆叠，操作按钮全宽。

## 2) 商户端-商品编辑页（多语字段录入）

### Page Structure
- 顶部：商品基础信息（既有）
- 中部：多语内容区（本次新增/改造核心）
- 底部：保存/发布操作条（既有）

### Sections & Components
1. 多语内容区 Header
   - 标题：多语言内容
   - 右侧：缺失提示汇总（Badge，例如“缺失 2 项”可点击展开）
2. 语言 Tab/Segmented Control
   - 三个 Tab：语言 A / 语言 B / 语言 C（展示真实语言名）
   - Tab 上展示状态点：
     - 绿色：该语言已完整填写
     - 橙色：该语言有缺失（允许缺失但提示）
     - 红色：默认语言缺失（阻止提交）
3. 多语字段表单（随 Tab 切换内容）
   - 字段范围：以现有商品模型为准（常见：名称、简介、详情）
   - 组件：Input / Textarea / 富文本编辑器（若现有已用）
   - 交互：
     - 切换 Tab 时自动保存草稿到本地 state
     - 离开页面前若有未保存变更，弹窗二次确认
4. 校验与提示
   - 默认语言：必填校验即时提示；保存/发布时强校验
   - 非默认语言：保存允许为空，但在 Header 汇总中提示缺失字段
5. 预览（可选，仅在现有已有预览能力时）
   - 若已有“预览商品”入口：预览中按当前语言 Tab 预览对应展示

### Responsive
- Desktop：表单 2 列（短字段）+ 详情字段独占整行。
- Tablet/Phone：单列堆叠；语言 Tab 固定在表单顶部。

## 3) MiniApp-设置/语言

### Page Structure
- 单列列表式设置页：顶部导航栏 + 设置项列表。

### Sections & Components
1. 顶部导航
   - 标题：语言
   - 返回按钮：返回上一页
2. 语言选择列表
   - 三个选项，单选（Radio List）
   - 选中态：右侧对勾；点击行即切换
3. 提示
   - 小字说明：语言会影响界面文案与商品展示语言；若内容缺失会自动回退。

### Responsive
- 以移动端为主；在大屏容器下列表居中，最大宽度 640px。
